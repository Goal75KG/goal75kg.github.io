真题7-1

*s和s区别

*s++：

  • *s++ 是一个组合表达式，首先会取出指针 s 指向的元素的值，然后再将指针 s 后移一个位置。

  • 执行顺序是先获取当前位置的值，然后再将指针自增。

示例：

int arr[] = {1, 2, 3, 4};
int *s = arr;
int value = *s++;
// value 的值为 1，然后 s 指向 arr[1]

s++：

  • s++ 是一个独立的自增表达式，只会将指针 s 后移一个位置，不会取出元素的值。

  • 执行顺序是先将指针自增，然后返回自增前的指针值。

示例：

int arr[] = {1, 2, 3, 4};
int *s = arr;
s++;
// 现在 s 指向 arr[1]

        c=a>b?a:b;
        return c;
}

        printf("%d", fun(5,4,0));
        return 0;
}
